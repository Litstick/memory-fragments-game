# 记忆碎片 - 在线H5解密游戏

通过拼凑记忆碎片、解开谜题来还原完整故事的解密游戏。

## 功能特性

- **拼图系统** — 图片拆分为 NxN 块，每块通过解谜解锁
- **6种谜题类型** — 文字问答、密码破解、凯撒密码、倒序文字、谜语、数学题
- **信封故事呈现** — 完成拼图后以信封拆开动画 + 打字机效果呈现故事
- **章节系统** — 多章节进度，每章独立拼图和故事
- **设置系统** — 自定义上传图片、编辑故事、配置谜题
- **存档功能** — 自动保存进度，支持继续游戏
- **预置示例** — 内置3章完整示例故事（27个谜题）

## 快速开始

直接用浏览器打开 `index.html` 即可，无需任何构建工具。

## 项目结构

```
├── index.html           # 主页面
├── css/
│   └── style.css        # 完整样式（暗色主题、动画、响应式）
└── js/
    ├── storage.js       # localStorage 存档系统
    ├── utils.js         # 工具函数
    ├── puzzle-types.js   # 谜题类型定义与验证
    ├── settings.js      # 设置页面与示例模板
    ├── chapter-edit.js  # 章节编辑
    ├── puzzle-edit.js   # 谜题编辑
    ├── game.js          # 游戏核心逻辑
    ├── puzzle.js        # 谜题交互
    ├── story.js         # 故事呈现
    └── main.js          # 入口初始化
```

## 使用说明

1. 打开 `index.html`
2. 点击 **设置** → **加载示例模板** 快速体验
3. 回主菜单点击 **开始新游戏**
4. 选择章节，点击模糊拼图块解开谜题
5. 全部解锁后观看故事呈现动画
